Clock Register Format (16-bit):
• Bits 0-11: Time in minutes (0-1439)
• Bits 12-14: Time ID (1-7)
• Bit 15: One Shot flag

Example: 5:00 AM with Time ID 1 = 300 minutes | (1 << 12) = 4396 = 0x112C → Protobuf: 78ac22
